About This Tool

Developed to provide an interface to monitor Staff Pool responses as they happen.

Administrator: Brendan Dagys

This site serves two purposes:

The first is to allow easy submission and recording of the staff that are assigned to posts at one of the Staff Pools, during a Code response. The three forms (see the menu bar) can be filled out and remove the need to radio up to the Command Centre with information.

The second purpose is to allow live tracking of staff assignments during a response. Using the Live Tracker, the Command Centre and other senior staff can see the names populate as staff are assigned.

Every entry into a form is automatically submitted to the database for future use.

Features

This site was designed to fulfill the following requirements:

Storage and Security

An encrypted database will store all results for future use, and pre-fills the form if more than one user is accessing it.

View on any device.

Responsive design means that you can submit/view using your device of preference.

Be Informed!

Once signed in, you can view the Staff Pool assignments in real-time as they occur. No need to refresh the page!
